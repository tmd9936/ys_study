블록레벨(block level)요소와 인라인(inline)요소

        1. div, h1, p
        2. 사용 가능한 최대 가로너비를 사용한다.test
        3. 크기지정할 수 있다.
        4. (width: 100%; height: 0으로 시작)
        5. 수직으로 쌓임
        6. matgin, padding을 상하좌우 사용가능하다.
        7. 레이아웃 설정시 많이 사용

        인라인요소
        1. span, img ... a ... 
        2. 필요한 만큼의 너비를 사용한다.
        3. 크기를 지정할 수 없다. (안의 내용에 따라 크기가 바뀜)
        4. (width: 0; height:0;으로 시작)
        5. 수평으로 쌓임
        6. margin padding 위, 아래는 사용할 수 없다
        7. Text와 관계있다.
